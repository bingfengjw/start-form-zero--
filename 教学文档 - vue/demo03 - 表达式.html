和angular一样，vue中也可以在{{}}中使用js的表达式（貌似在使用js表达式方面比angular更加智能）

使用v-bind:class和v-bind:id可以动态生成class和id

12+20 = {{12+20}}
12-20 = {{12-20}}
12*20 = {{12*20}}
12/20 = {{12/20}}
num+20 = {{num1+20}}
num1+num2 = {{num1+num2}}
str1+(num1+num2)+str2 = {{str1+(num1+num2)+str2}}
a?'yes':'no' = {{a?'yes':'no'}}
b.split('').reverse().join('') = {{b.split('').reverse().join('')}}
动态生成class
动态生成id
